# PR-212 — Free shipping threshold

## Goal

Add per-zone free shipping threshold so that when cart subtotal meets or
exceeds the configured amount, a zero-cost "free_shipping" rate appears
alongside normal paid rates. Implements SHIPPING_ZONES.md §4.

## Design notes

1. **FreeShippingThreshold on Zone** — `shared.Money` value. Zero amount
   means disabled. Stored as two DB columns: `free_shipping_threshold`
   (BIGINT, CHECK >= 0) and `free_shipping_currency` (TEXT).

2. **CalculateRate returns []ShippingRate** — Breaking change from single
   return. When threshold is met, slice contains paid rate + free rate.
   When not met, single-element slice with paid rate only.

3. **qualifiesForFreeShipping** — Checks three conditions: threshold
   amount > 0, threshold currency matches request currency (EqualFold),
   subtotal amount >= threshold amount.

4. **Free rate label** — Format: `"Free shipping (orders over {amount} {currency})"`.

5. **Admin CRUD** — `free_shipping_threshold` is optional on create
   (nil = 0). On update, setting to 0 clears threshold. Currency
   defaults to "EUR" when not specified.

6. **MethodFreeShipping** — New shipping method constant added to
   `IsValid()` alongside `flat_rate` and `weight_based`.
