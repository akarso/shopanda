# PR-202: docker-compose.yml

## Summary

Docker Compose file for local development and deployment. Default profile runs the app + Postgres; optional profiles add Mailpit (email testing) and Meilisearch (full-text search).

## Changes

### New Files

- **`docker-compose.yml`** — Compose configuration with:
  - **`app`**: builds from local Dockerfile, maps `SHOPANDA_SERVER_PORT` (default 8080), reads `.env` (optional — won't error if missing), waits for healthy Postgres. Sets `SHOPANDA_DATABASE_HOST=postgres` and `SHOPANDA_MAIL_SMTP_HOST=mailpit`/`PORT=1025` so the container reaches sibling services without relying on `.env` overrides
  - **`postgres`**: PostgreSQL 17 Alpine (pinned by digest), credentials from `.env` with sensible defaults, named volume for data persistence, built-in health check via `pg_isready` with `start_period: 10s`
  - **`mailpit`** (profile: `dev`): local SMTP sink on port 1025, web UI on port 8025 — use with `SHOPANDA_MAIL_SMTP_HOST=mailpit` and `SHOPANDA_MAIL_SMTP_PORT=1025`
  - **`meilisearch`** (profile: `search`): Meilisearch v1.13 (pinned by digest) on localhost:7700, named volume for index persistence, analytics disabled, intentionally no `MEILI_MASTER_KEY` (local dev only)
  - All third-party images pinned by `@sha256:` digest for reproducible builds
  - Mailpit and Meilisearch ports bound to `127.0.0.1` (loopback only)

## Usage

```bash
# Default: app + postgres
docker compose up -d

# Run migrations
docker compose run --rm app migrate

# Seed data
docker compose run --rm app seed

# With Mailpit for email testing
docker compose --profile dev up -d

# With Meilisearch for full-text search
docker compose --profile search up -d

# All services
docker compose --profile dev --profile search up -d
```

## Profiles

| Profile   | Adds          | Ports (loopback)                    |
| --------- | ------------- | ----------------------------------- |
| (default) | app + postgres | 8080                               |
| `dev`     | mailpit       | 127.0.0.1:8025 (UI), 127.0.0.1:1025 (SMTP) |
| `search`  | meilisearch   | 127.0.0.1:7700                      |
