# PR-202: docker-compose.yml

## Summary

Docker Compose file for local development and deployment. Default profile runs the app + Postgres; optional profiles add Mailpit (email testing) and Meilisearch (full-text search).

## Changes

### New Files

- **`docker-compose.yml`** — Compose configuration with:
  - **`app`**: builds from local Dockerfile, maps `SHOPANDA_SERVER_PORT` (default 8080), reads `.env` (optional — won't error if missing), waits for healthy Postgres
  - **`postgres`**: PostgreSQL 17 Alpine (pinned by digest), credentials from `.env` with sensible defaults, named volume for data persistence, built-in health check via `pg_isready`
  - **`mailpit`** (profile: `dev`): local SMTP sink on port 1025, web UI on port 8025 — use with `SHOPANDA_MAIL_SMTP_HOST=mailpit` and `SHOPANDA_MAIL_SMTP_PORT=1025`
  - **`meilisearch`** (profile: `search`): Meilisearch v1.13 (pinned by digest) on port 7700, named volume for index persistence, analytics disabled
  - All third-party images pinned by `@sha256:` digest for reproducible builds

## Usage

```bash
# Default: app + postgres
docker compose up -d

# Run migrations
docker compose run --rm app migrate

# Seed data
docker compose run --rm app seed

# With Mailpit for email testing
docker compose --profile dev up -d

# With Meilisearch for full-text search
docker compose --profile search up -d

# All services
docker compose --profile dev --profile search up -d
```

## Profiles

| Profile   | Adds          | Ports                    |
| --------- | ------------- | ------------------------ |
| (default) | app + postgres | 8080                    |
| `dev`     | mailpit       | 8025 (UI), 1025 (SMTP)  |
| `search`  | meilisearch   | 7700                     |
