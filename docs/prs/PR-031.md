## Schema

```sql
CREATE TABLE customers (
    id            UUID PRIMARY KEY,
    email         TEXT UNIQUE NOT NULL,
    first_name    TEXT NOT NULL DEFAULT '',
    last_name     TEXT NOT NULL DEFAULT '',
    password_hash TEXT NOT NULL DEFAULT '',
    status        TEXT NOT NULL DEFAULT 'active' CHECK (status IN ('active', 'disabled')),
    created_at    TIMESTAMPTZ NOT NULL DEFAULT now(),
    updated_at    TIMESTAMPTZ NOT NULL DEFAULT now()
);
```

## Entity fields

| Field | Type | Notes |
|-------|------|-------|
| ID | string (UUID) | Generated by caller via `id.New()` |
| Email | string | Required, unique |
| FirstName | string | Optional, defaults to empty |
| LastName | string | Optional, defaults to empty |
| PasswordHash | string | Set via `SetPassword()`; populated by PR-032 |
| Status | Status | `active` (default) or `disabled` |
| CreatedAt | time.Time | Set on creation |
| UpdatedAt | time.Time | Updated on mutations |
