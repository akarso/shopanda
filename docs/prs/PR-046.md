# PR-046 — Flat Rate Shipping Provider

## Summary

Adds the built-in flat rate shipping provider and a public `GET /api/v1/shipping/rates` endpoint that returns available shipping rate quotes.

## Design

### Flat rate provider

- Implements `shipping.Provider` interface with compile-time check
- Configured with a fixed `shared.Money` cost at construction time
- Returns the configured cost for any order in matching currency
- Rejects requests for non-matching currencies (provider is skipped by handler)
- Provider ref format: `flat_rate:<orderID>`
- Default configuration: 500 cents USD ($5.00)

### Shipping rates endpoint

```text
GET /api/v1/shipping/rates?order_id=X&currency=USD&item_count=3
```

- Behind `RequireAuth` (customer-facing, used during checkout)
- Iterates all registered providers, collects rates
- Silently skips providers that return errors (e.g. currency mismatch)
- Returns `{"data":{"rates":[...]}}` with method, provider_ref, cost, currency, label

## Test coverage

- **Provider**: method, calculate rate, currency mismatch, item count invariance
- **Handler**: success, missing order_id, missing currency, missing item_count, invalid item_count (3 sub-cases), provider error skipped, panic on no providers
