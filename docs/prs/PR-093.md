## Design

### Sitemap

- Paginates through products (page size 100), filters to `StatusActive` only
- Fetches all categories via `FindAll`
- Paginates through CMS pages, filters to `IsActive()` only
- Emits standard XML sitemap (`http://www.sitemaps.org/schemas/sitemap/0.9`) with `encoding/xml`
- Each `<url>` has `<loc>` (absolute URL) and `<lastmod>` (date only, `2006-01-02`)

### Robots

- Static response: `User-agent: *`, `Allow: /`, `Sitemap: {baseURL}/sitemap.xml`
- Body computed once at handler creation, not per request

### Canonical URLs

- `CanonicalURLStep`: sets `ctx.Meta["canonical"]` to `{baseURL}/products/{slug}` for PDP
- `ListingCanonicalURLStep`: sets `ctx.Meta["canonical"]` to `{baseURL}/products` for PLP
- Both guard against nil ctx and nil/empty product
