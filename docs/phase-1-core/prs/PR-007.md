# PR-007 — HTTP Server Skeleton

## Summary

Adds the HTTP server layer: router with middleware chain, health endpoint, graceful shutdown, and three core middleware (recovery, request-ID, logging). Default command is now `serve`.

| `internal/interfaces/http/health.go` | New — `GET /healthz` returning `{"status":"ok"}` |
| `internal/interfaces/http/server.go` | New — `Server` with `ListenAndServe()`, graceful shutdown on SIGINT/SIGTERM |

## Public API

```go
// Router
func NewRouter() *Router
func (r *Router) Use(mw ...Middleware)
func (r *Router) Handle(pattern string, handler http.Handler)
func (r *Router) HandleFunc(pattern string, handler http.HandlerFunc)
func (r *Router) Handler() http.Handler

// Server
func NewServer(host string, port int, handler http.Handler, log logger.Logger) *Server
func (s *Server) ListenAndServe() error
func (s *Server) Addr() string

// Middleware
func RequestIDMiddleware() Middleware
func LoggingMiddleware(log logger.Logger) Middleware
func RecoveryMiddleware(log logger.Logger) Middleware

// Health
func HealthHandler() http.HandlerFunc
```

## Design Decisions

- **`net/http` standard library** — Go 1.22+ enhanced `ServeMux` supports method-based routing (`GET /healthz`). No external router dependency needed.
- **Middleware as `func(http.Handler) http.Handler`** — standard Go pattern, composable, no framework.
- **Graceful shutdown** — `SIGINT`/`SIGTERM` → 15s drain → `srv.Shutdown()`. Clean connection handling.
- **Server timeouts** — Read: 10s, Write: 30s, Idle: 60s. Sensible defaults.
- **Default command = serve** — `go run ./cmd/api/` starts the server. `go run ./cmd/api/ serve` is explicit alias.
- **Recovery middleware** — catches panics, logs them, returns 500. Prevents server crashes from handler panics.
- **Middleware order** — Recovery (outermost) → RequestID → Logging (innermost before handler). Logging captures request-ID set by middleware above it.
