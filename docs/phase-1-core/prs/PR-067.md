# PR-067 ŌĆö Product Admin Schema

## Summary

Registers product form and grid schemas with the admin registry and exposes them via schema-driven API endpoints.

## Changes

### New Files

- **`internal/application/admin/product_schema.go`** ŌĆö `RegisterProductSchemas(r *admin.Registry)` registers:
  - `product.form`: 4 fields (name, slug, description, status with select options)
  - `product.grid`: 6 columns (id, name, slug, status, created_at, updated_at)

- **`internal/interfaces/http/schema_handler.go`** ŌĆö `SchemaHandler` with JSON-safe DTO serialization:
  - `GET /api/v1/admin/forms/{name}` ŌĆö returns form schema
  - `GET /api/v1/admin/grids/{name}` ŌĆö returns grid schema
  - Converts domain types to DTOs that omit non-serializable `func` fields (`Column.Value`, `Action.Execute`)

- **`internal/application/admin/product_schema_test.go`** ŌĆö 2 tests verifying form fields and grid columns
- **`internal/interfaces/http/schema_handler_test.go`** ŌĆö 6 tests: form OK, form not found, grid OK, grid not found, form requires admin, grid requires admin

### Modified Files

- **`cmd/api/main.go`** ŌĆö Wires `admin.NewRegistry()`, calls `RegisterProductSchemas`, creates `SchemaHandler`, registers admin routes
- **`docs/diagrams/c4-component.md`** ŌĆö Added `SchemaHandler` and `ProductSchemaRegistration` components with relationships
- **`docs/diagrams/c4-code.md`** ŌĆö Added `SchemaHandler` class with `AdminRegistry` relationship

## API Endpoints

| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | `/api/v1/admin/forms/{name}` | admin | Returns form schema by name |
| GET | `/api/v1/admin/grids/{name}` | admin | Returns grid schema by name |

### Response Example (form)

```json
{
  "data": {
    "form": {
      "name": "product.form",
      "fields": [
        {"name": "name", "type": "text", "label": "Product Name", "required": true},
        {"name": "status", "type": "select", "label": "Status", "default": "draft", "options": [
          {"label": "Draft", "value": "draft"},
          {"label": "Active", "value": "active"},
          {"label": "Archived", "value": "archived"}
        ]}
      ]
    }
  },
  "error": null
}
```

## Design Decisions

- **DTO layer for JSON serialization**: `Column.Value` (func) and `Action.Execute` (func) cannot be marshaled to JSON. Handler converts domain types to JSON-safe DTOs that expose only serializable data.
- **Schema endpoints are admin-only**: Protected by `RequireRole(admin)` middleware.
- **Registry wired at startup**: Product schemas registered once during `main.go` initialization, before the server starts.
