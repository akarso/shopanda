# PR-002: Platform — config loader

## Phase
Phase 0 — Project Scaffold

## Scope
`config.yaml` parsing, env overlay, `config.Get()`

## Files Created
- `internal/platform/config/config.go` — `Config` struct (Server, Database, Log), YAML parsing via `gopkg.in/yaml.v3`, `SHOPANDA_*` env overlay, `Get`/`GetString`/`GetInt`/`GetFloat` dot-notation accessors, `DatabaseDSN`, `FindConfigFile`, redacted `String()`
- `internal/platform/config/config_test.go` — 11 tests: defaults, YAML overrides, env overrides, dot-notation access, DSN, file-not-found, invalid YAML, password redaction

## Files Modified
- `cmd/api/main.go` — wired `config.Load(config.FindConfigFile())` into `run()`
- `go.mod` / `go.sum` — added `gopkg.in/yaml.v3 v3.0.1`

## Dependencies Added
- `gopkg.in/yaml.v3` — YAML parsing (no stdlib alternative)

## Key Decisions
- `SHOPANDA_` prefix for env vars to avoid collisions
- `DATABASE_URL` env var takes precedence over config struct DSN
- Flat dot-notation map (`server.port`, `database.host`) for generic access
- DB config layer deferred to PR-064

## Verification
- `go test ./internal/platform/config/` — 11/11 pass
- `go build ./cmd/api/` — compiles
- `go vet ./...` — passes
- `go run ./cmd/api/` — loads config, prints redacted summary
